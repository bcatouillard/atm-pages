# Productive Trees — Débuter

Productive Trees permet de cultiver des arbres.

Points importants pour débuter :

- Avoir une rûche avec des abeilles à proximité pour que les arbres poussent. N'oubliez pas de mettre des fleurs à proximité pour qu'elles puissent avoir du pollen.
- Placer les arbres proches entre eux pour favoriser la pollinisation croisée et l'amélioration génétique.

# Conseils pratiques

## Pollen upgrade dans la rûche

L'upgrade de pollen dans la rûche permet d'obtenir du pollen dans la rûche est de l'appliquer soit même sur les feuilles.

Pour cela il vous faudra avoir une Advanced beehive pour mettre des upgrades.
