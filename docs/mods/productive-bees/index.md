# Productive Bees — Aperçu

`Productive Bees` permet d'obtenir des ressources via l'apiculture et la manipulation génétique des abeilles. Points importants :

- Utiliser des extracteurs/automates pour récupérer les ressources et les envoyer vers le stockage central.
- Documenter les espèces utiles et leurs gènes sur cette page pour faciliter le partage entre joueurs.

Voir la page "Génération de ressources (gènes)" pour des conseils sur l'optimisation.
