# Génération de ressources — Gènes (Productive Bees)

Pour maximiser la génération de ressources via `Productive Bees`, concentrez-vous sur :

- L'amélioration des gènes liés à la production (productivity, weather tolerance et behavior).

Il vous faudra plusieurs de ces abeilles pour avoir un gène de chaque à 100%. Voici les étapes pour les obtenir :

# Productivity: Very High

- Augmente la production de ressources par les abeilles.

Equipez un Bee nest helmet et faites vous taper par des mobs. Vous verrez des KamikazBees apparaître, capturez les et écrasez le sur le bottler pour extraire les gênes.

# Weather Tolerance: High

- Permet aux abeilles de continuer à produire même en cas de mauvais temps (pluie, neige).

Pour obtenir ce gène, il vous faudra pêcher dans un biome River pour avoir une chance de voir des Water Bees apparaître. Capturez les et écrasez les sur le bottler pour extraire les gênes.

# Behavior: Metaturnal

- Les abeilles métaturnales produisent des ressources à la fois le jour et la nuit, augmentant ainsi leur efficacité globale.

Pour l'obtenir, prenez des abeilles et accrochez les à une fence avec une lead. Ensuite, attendez que le jour et la nuit passe pour les voir devenir Nocturnal puis Metaturnal.

Ajoutez ici des captures d'écran des interfaces de croisement et des panneaux de gènes.

# Bottler

Le bottler est l'outil principal pour extraire les gènes des abeilles. Placez une abeille sur le bottler et appuyez sur le bouton pour extraire les gènes. Vous pouvez ensuite utiliser la bouteille que vous venez d'obtenir et la mettre dans une centrifugeuse pour avoir les gènes.

# Gene Indexer

Le gene indexer est l'outil qui fusionnera les gènes pour vous. Placez les gènes extraits dans le bottler et ajoutez les dedans.

# Conclusion

Pour donner ces gènes à une abeille, prenez une Honey treat et combinez la avec les gènes qui vous intéressent. Une fois fait, donnez cette Honey treat à une abeille pour lui transmettre les gènes.
