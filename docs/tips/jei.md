# JEI — Tips rapides

JEI (Just Enough Items) est l'outil principal pour consulter recettes et usages :

- Filtrer par mod : activez le filtre par mod pour ne voir que les recettes d'un mod donné.
- Voir les recettes/usages : sélectionnez un item et appuyez sur la touche recette pour afficher toutes les façons de le fabriquer.
- Noter les recettes complexes : utilisez la touche de favoris pour garder en mémoire les recettes importantes.

## Voir les recettes

Sélectionnez un item dans JEI et appuyez sur la touche de recette (par défaut `R`) pour afficher toutes les recettes qui permettent de fabriquer cet item. Cela vous aidera à comprendre les composants nécessaires et les machines impliquées.

Appuyez sur `R` pour voir les recettes de fabrication, ou sur `U` pour voir les usages d'un item. Vous pouvez naviguer entre les différents onglets de recettes (Crafting, Smelting, Machine X, etc.) pour trouver celle qui correspond à votre besoin.

## Filtrer par mod

Utilisez les filtres sur le côté pour n'afficher que les recettes d'un mod spécifique. Cela vous aidera à trouver rapidement les machines et composants liés à ce mod sans être submergé par les autres recettes.

Exemple : si vous cherchez des recettes de `Productive Bees`, utilisez `@Productive Bees` pour n'afficher que les recettes de ce mod.

## Noter les recettes complexes

Pour les crafts complexes (ex : machines avec plusieurs composants, ou recettes nécessitant des étapes intermédiaires), vous pouvez noter les recettes et les mettre de côté en appuyant sur la touche de favoris (par défaut `Q`). Cela vous permettra de les retrouver facilement plus tard sans devoir les chercher à nouveau.
